## Running Locally

```bash
git clone git@github.com:benjamin-chavez/benjamin-chavez.com.git
cd benjamin-chavez.com
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the locally running site.
